How project works

There is a load balancing loadbalancer which is used to divide the load equally among various servers. The client sends the request to the loadbalancer which checks the conditions on the sarvers and assings the request to loadbalancer with the least amount of load. There is a server function present in the loadbalancer which processes the request from the user and sends the result. The aim of the project is to prevent crash due to DDOS.
